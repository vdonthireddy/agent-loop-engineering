# Calculator Application

## Overview
A simple command-line calculator written in Python that can perform basic arithmetic operations.

## Features
- Addition, subtraction, multiplication, and division of two numbers.
- Handles division by zero gracefully by returning an error message.
- Simple, stateless function-based design.

## Non-Functional Requirements
- Must be written in Python.
- Must include unit tests using pytest.
- Must include a `smoke_check.sh` that invokes the calculator script with a sample calculation to prove it runs.
