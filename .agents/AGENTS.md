
## Artifact Location Rule
Always create artifact files (such as `implementation_plan.md`, `task.md`, and `walkthrough.md`) directly in the workspace root directory, NOT in the default `<appDataDir>/brain/<conversation-id>/` directory.

## Core Philosophical Alignment
The primary focus of this project is **Loop Engineering**—the science and tooling of designing, orchestrating, and optimizing generic agentic feedback loops. You must NOT treat any specific architecture (like Actor-Critic) as the main focus or final product. The framework is a loop engineering laboratory, and any architecture implemented is merely a proof-of-concept configuration of a loop. All your suggestions, code, and documentation must serve the meta-goal of making loop design easier and more robust.
